# Node Functions

## 概述

EdgeOne Node Functions 后端代码。

## 路由映射

根据 EdgeOne Node Functions 的路由规则，以下路由将被映射：

| 文件路径 | 访问路径 |
|---------|---------|
| `/node-functions/express/[[default]].js` | `/express/` |
| `/node-functions/test.js` | `/test` |

## 环境变量

环境变量统一在项目根目录的 `README.md` 中维护，请以根目录文档为准。

## 本地联调

Node Functions 会在 EdgeOne Pages 本地开发模式下运行：

```bash
edgeone pages dev
```
